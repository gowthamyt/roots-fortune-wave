# Roots Cosmo Private Limited

Corporate website for Roots Cosmo Private Limited, built with TanStack Start, React, TypeScript, Tailwind CSS, and Nitro.

## Development

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
```

## Netlify

This project is configured for Netlify with `netlify.toml`.

- Build command: `npm run build`
- Publish directory: `dist`
- Runtime: Node.js 22
